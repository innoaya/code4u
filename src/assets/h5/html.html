HTML ဆိုတာဘာလဲ?

HTML ဆိုတာ HyperText Markup Language ဖြစ်ပါတယ်။ website တိုင်းရဲ့ အခြေခံအဆောက်အဦပါ!

HTML မှာ tags တွေကို သုံးပြီး webpage အစိတ်အပိုင်းတွေ ဖန်တီးကြတယ်။

ဥပမာ:

စသင်ကြရအောင်! 🚀

🏗️ HTML Page အခြေခံဖွဲ့စည်းပုံ

HTML document တိုင်းမှာ ဒီအစိတ်အပိုင်းတွေ ပါဝင်ပါတယ်:

HTML အခြေခံဖွဲ့စည်းပုံ

Tags အချို့ကို မိတ်ဆက်ခြင်း:

<!DOCTYPE html>

ဒါက HTML5 ဖြစ်ကြောင်း ပြောတာပါ

<html>

အရာအားလုံးရဲ့ အခြေခံအိုးပါ

<head>

မြင်ရတဲ့အချက်အလက် (title လိုမျိုး)

<body>

မြင်ရတဲ့အကြောင်းအရာတွေ ထည့်ရာနေရာ

<title>

Browser tab မှာ ပြမယ့်ခေါင်းစဉ်

<meta>

စာမျက်နှာအကြောင်း အချက်အလက်

🛠️ ပထမဆုံး စိန်ခေါ်မှု!

ဒီလိုစာမျက်နှာလေး တည်ဆောက်ကြည့်ပါ:

  1. ခေါင်းစဉ်တစ်ခု ("ကျွန်တော့်အိမ်မွေးတိရစ္ဆာန်")
  2. သင့်အိမ်မွေးတိရစ္ဆာန်အကြောင်း စာပိုဒ်တိုလေး

HTML မှာ ဒါတွေကို စမ်းကြည့်ပါ:

စာပိုဒ်

<br>

စာကြောင်းခွဲ (အလိုအလျောက်ပိတ်)

<p>စာကြောင်း 1<br>စာကြောင်း 2</p>
<hr>

အလျားလိုက်မျဉ်း (အလိုအလျောက်ပိတ်)

<pre>

ကြိုတင်ဖော်မတ်ထားသော စာသား (နေရာလွတ်များကို ထိန်းသိမ်း)

<blockquote>

အတုအယောင် ကိုးကားချက်

🏛️ Semantic HTML

Browser နဲ့ screen reader တွေအတွက် ၎င်းတို့၏ရည်ရွယ်ချက်ကို ဖော်ပြတဲ့ modern tags တွေ:

Semantic Layout ဥပမာ:

Semantic Tag အဖွဲ့:

<header>

မိတ်ဆက်အကြောင်းအရာ (စာမျက်နှာ သို့မဟုတ် အပိုင်းခေါင်းစဉ်)

<nav>

သွားလာရန် လင့်ခ်များ

<main>

အဓိက အကြောင်းအရာ (စာမျက်နှာတစ်ခုလျှင် တစ်ခုသာ)

<article>

သီးခြားဖွဲ့စည်းမှု (blog post, သတင်းအကြောင်းအရာ)

<section>

အကြောင်းအရာ၏ အပိုင်းအစများ

<aside>

ဘေးထွက် အကြောင်းအရာ (sidebar ကဲ့သို့)

<footer>

စာမျက်နှာ သို့မဟုတ် အပိုင်းအတွက် အောက်ခြေစာသား

သိပါသလား? Semantic HTML က ရှာဖွေရေးအင်ဂျင်တွေကို သင့်အကြောင်းအရာကို ပိုနားလည်စေပြီး website ranking ကို မြှင့်တင်နိုင်ပါတယ်!
