HTML & CSS ဆိုတာဘာလဲ?

CSS - Website ၏ အလှဆင်မှုနှင့် အပြင်အဆင် (အရောင်နှင့်ဒီဇိုင်း)

🎨 CSS အခြေခံများ

CSS ကို HTML မှာ ၃နည်းနဲ့ သုံးနိုင်ပါတယ်:

၁. Inline CSS (HTML tag တွင်တိုက်ရိုက်)

၂. Internal CSS (head ထဲတွင် style tag ဖြင့်)

၃. External CSS (သီးသန့် .css ဖိုင်ဖြင့်)

အကောင်းဆုံးအလေ့အကျင့်: External CSS ကိုသုံးပါ၊ ဘာကြောင့်လဲဆိုတော့ ပြန်အသုံးပြုရလွယ်ပြီး စီမံရလွယ်ကူပါတယ်!

🔍 CSS Selectors များ

HTML elements တွေကို ဘယ်လိုရွေးချယ်မလဲ?

အဓိက Selector အမျိုးအစားများ:

Element Selector
p { color: red; }

tag name ဖြင့် ရွေးချယ်

Class Selector
.intro { }

class attribute ဖြင့် ရွေးချယ်

ID Selector
#header { }

id attribute ဖြင့် ရွေးချယ်

Grouping Selector
h1, h2, h3 { }

တစ်ပြိုင်နက် ရွေးချယ်

Descendant Selector
div p { }

အတွင်းပိုင်း element ရွေးချယ်

Pseudo-class
a:hover { }

အခြေအနေအလိုက် ရွေးချယ်

🛠️ Selector စိန်ခေါ်မှု!

ဒီ HTML အတွက် CSS selector တွေရေးပါ:

  1. "intro" class ရှိ p tag ကို အပြာရောင်လုပ်ပါ
  2. list item တွေရဲ့ ပထမဆုံးအချက်ကို စာလုံးထူးပါ
  3. footer id ရှိ div ကို နောက်ခံမီးခိုးရောင်လုပ်ပါ

✏️ စာသားနှင့် ဖောင့်များ

စာသားတွေကို အလှဆင်ကြမယ်!

စာသားအလှဆင်နည်းများ:

font-family

စာလုံးအမျိုးအစား

font-size

စာလုံးအရွယ်အစား

font-size: 16px;
font-weight

စာလုံးထု

font-weight: bold;
text-align

စာသားညှိနှိုင်းမှု

text-align: center;
line-height

စာကြောင်းအကြား အကွာအဝေး

line-height: 1.6;
text-shadow

စာသားအရိပ်

📐 Layout နှင့် Positioning

Element တွေကို ဘယ်လိုနေရာချမလဲ?

Positioning အမျိုးအစားများ:

display

Element ၏ display type

display: block | inline | flex | grid;
position

Element ၏ positioning method

position: static | relative | absolute | fixed | sticky;
top, right, bottom, left

Positioned elements အတွက် အကွာအဝေး

top: 10px; left: 20px;
z-index

Element ၏ အလွှာအဆင့်

z-index: 10;
float

Element ကို ဘေးဘက်သို့ ပေါ်စေ

float: left | right;

🛠️ Layout စိန်ခေါ်မှု!

ဒီ layout ကို ဖန်တီးပါ:

  1. Header (fixed position, top 0, full width)
  2. Sidebar (left side, 200px width)
  3. Main content (sidebar နဲ့ 20px ကွာ)
  4. Footer (bottom မှာ sticky)
