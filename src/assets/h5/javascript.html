Data Types အဓိက ၇မျိုး

  1. String (စာသား) - "Hello"
  2. Number (ဂဏန်း) - 123, 3.14
  3. Boolean (အမှန်/အမှား) - true/false
  4. Array (စာရင်း) - [1, 2, 3]
  5. Object (အရာဝတ္ထု) - {name: "John"}
  6. Null (အချည်းနှီး)

Variable Types ၃မျိုး

  • let - Block scope, reassignable
  • const - Block scope, constant
  • var - Function scope, outdated

အကြံပြုချက်: const ကိုအများဆုံးသုံးပါ၊ လိုအပ်မှ let သုံးပါ၊ var ကိုမသုံးပါနဲ့
